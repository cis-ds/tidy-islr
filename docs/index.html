What this is about

This book is intended to serve as a companion to An Introduction to Statistical Learning with Applications in R (ISLR) and reimplement the R programs using primarily tidyverse packages and a tidy design philosophy. It is not intended to stand on its own or review the statistical learning concepts from ISLR. It is not intended to teach R programming or computational problem solving in general.
